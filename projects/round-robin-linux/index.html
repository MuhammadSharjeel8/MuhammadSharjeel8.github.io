Round-Robin Scheduling Model in C for Linux Environments

by Muhammad Sharjeel

ESE 333: Real-Time Operating Systems

Project Overview

This project implements a simulation of a round-robin CPU scheduling algorithm using the C programming language. It models how modern operating systems distribute CPU time across multiple cores using time-sliced multitasking. The simulation also handles process arrivals, service time tracking, and I/O interruptions (in Part 3).

Features

  • Round-robin scheduling with a configurable time quantum
  • Multi-core (4-core) CPU simulation
  • Linked listâ€“based queue management
  • I/O event handling
  • Accurate time and process state tracking
  • Designed to run on Linux systems

Getting Started

Compilation

gcc -o sched sched.c

Execution

./sched

This project was tested in Linux environments using both native systems and virtual machines, including:

  • Ubuntu ARM64 via VirtualBuddy on macOS (M3)
  • Ubuntu via WSL on Windows 11
