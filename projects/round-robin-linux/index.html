Muhammad Sharjeel

Simulation of CPU round-robin scheduling on a 4-core processor

View the Private GitHub Repository

Project Overview

This project implements a round-robin CPU scheduling algorithm in C designed to simulate a 4-core CPU environment running on Linux systems. It demonstrates fundamental operating system concepts such as multitasking, process scheduling, and I/O interrupt handling.

Features

  • Round-robin scheduling with configurable time quantum
  • Multi-core (4 cores) CPU simulation
  • Linked list–based process queue management
  • I/O interrupt event simulation and handling
  • Designed and tested for Linux environments

How to Run

To compile and run the program on a Linux system or WSL:

gcc -o sched sched.c
./sched
