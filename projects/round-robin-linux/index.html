Project Description

This project implements a simulation of the round-robin CPU scheduling algorithm in C, modeling how modern Linux-based operating systems allocate CPU time using time-sliced multitasking across multiple cores. The implementation includes linked-list–based queue management, I/O event handling, and detailed state tracking.

Assignment Summary

  • Simulates multi-core round-robin scheduling with configurable quantum
  • Manages processes with arrival times and I/O events
  • Implements process queues with linked lists
  • Handles process state transitions and memory cleanup

Design and Implementation

The scheduler simulates four CPU cores, dispatching processes according to round-robin rules. Processes are queued, scheduled, and preempted based on quantum expiration or I/O events, with service times properly decremented and memory managed dynamically.

File Structure

sched.c
Main C source file containing the entire scheduling simulation, including all structs and functions.
input.txt
Input file specifying processes’ IDs, service times, and arrival times.

How to Compile and Run

gcc -o sched sched.c
./sched
