Round-Robin Scheduling Model in C

Muhammad Sharjeel

ESE 333: Real-Time Operating Systems

Project Overview

This project implements a simulation of the round-robin CPU scheduling algorithm in C, modeling how modern Linux-based operating systems allocate CPU time using time-sliced multitasking across multiple cores. The implementation includes linked-list–based queue management, I/O event handling, and detailed state tracking.

Platform & Tools

  • Tested on Ubuntu via WSL (Windows 11)
  • Also tested on Ubuntu ARM64 via VirtualBuddy (macOS M3)
  • Written in C and compiled with GCC

Key Features

  • Multi-core simulation (4 cores)
  • Round-robin scheduler with configurable time quantum
  • Support for I/O blocking and process state transitions
  • Accurate simulation of real-time system behavior

How to Run

gcc -o sched sched.c
./sched
