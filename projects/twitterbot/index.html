Project Description

This project involves building a Twitterbot using Python and the simple_twit library (a simplified wrapper for Tweepy) to interact with Twitter’s API. The bot performs automated interactions such as retrieving tweets from timelines and posting tweets (including media).

Assignment Summary

  • Created a unique Twitter account for the project using assigned credentials
  • Used simple_twit for all Twitter API interactions
  • Completed tasks like:
    • Retrieve and print 3 most recent tweets from the bot’s home timeline
    • Retrieve and print 5 most recent tweets from another user
    • Post a single tweet and a media tweet
  • Developed a Twitterbot that posts randomized quotes
  • Followed Twitter’s automation rules responsibly

Design and Implementation

The bot randomly selects quotes from a predefined list and posts them. It also demonstrates interaction capabilities including reading timelines, tweeting text, and posting media content.
