Project Description

This project is a Python-based Twitterbot that posts tweets automatically using the Tweepy library. It uses OAuth for secure API access, formats tweet content dynamically, and supports both scheduled and manual posting.

Technologies Used

  • Python 3
  • Tweepy (Twitter API)
  • OAuth 1.0a authentication
  • Scheduling via time and threading

Features

  • Secure OAuth login and key management
  • Custom tweet formatting with hashtags
  • Support for periodic or event-triggered tweets
  • Error handling and status logging

Sample Output

Tweet sent at 08:00: 'Good morning, world! ☀️ #MondayMotivation'
